# RutValidator

Chilean RUT validator for ActiveRecord on rails 3.

## Installation

Add this line to your application's Gemfile:

    gem 'rut_validator'

Run:

    $ bundle

## Usage

Add the following to your model:

    validates :rut_attribute, rut: true
